Խորացված JavaScript առցանց անվճար դասընթաց

Մանրամասներ

Դասընթացը տեսաձայնագրվում է կամ գտնվում է տեսաձայնագրման հերթի մեջ։ Բաց չթողնելու համար բաժանորդագրվեք Code Republic-ի ալիքին YouTube-ում։

Ծրագրավորման հարցազրույցների պատրաստում (5 ամիս)

Դասընթացի նպատակն է ինտենսիվ պատրաստել Ձեզ ծրագրավորման հարցազրույցները հաջողությամբ հաղթահարելու համար։ Ալգորիթմներ և տվյալների կառուցվածքներ, համակարգային նախագծում (OOP, design patterns), տվյալների բազաներ, ծրագրավորման լեզուներ, անգլերեն փորձնական հարցազրույցներ և մի շարք այլ թեմաներ։ Դասընթացի ավարտից հետո ուսանողները մասնակցում են 3 ամիս տևողությամբ անվճար պրակտիկ նախագծի իրականացմանը ամբողջական աշխատանքային պորտֆոլիո ունենալու համար։

Computer Science և ծրագրավորման հիմունքներ սկսնակների համար (6 ամիս)

Ծրագրավորման դասընթաց սկսնակների համար, որը սովորեցնում է ծրագրավորման հիմունքները, համակարգչային համակարգերի կազմակերպում, խնդիրներ լուծելու մեթոդներ, ալգորիթմների և տվյալների կառուցվածքների կիրառություն, C++ ծրագրավորման լեզուն խորությամբ, ինչպես նաև JavaScript ծրագրավորման լեզվի հիմունքներ։ Խորհուրդ է տրվում մինչև դիմելը դիտել Ծրագրավորում բացարձակ սկսնակների համար առցանց դասընթացը։ Դասընթացի 3րդ ամսից ուսանողները մասնակցում են պրակտիկ նախագծերի վրա աշխատանքներին։ Ավարտելուց հետո ևս երեք ամիս աշխատում են պրակտիկ փորձ ձեռք բերելու համար։

Վեբ ծրագրավորման խորացված դասընթաց (Backend & DevOps Beast) (8 ամիս)

Յուրահատուկ դասընթաց, որն աշխատանքում հանդիպող լեզուները, տեխնոլոգիաներն ու հասկացությունները խորացված ուսումնասիրելուն զուգընթաց շեշտադրում է իրական նախագծերի վրա պրակտիկ աշխատանք։ Backend վեբ ծրագրավորման մեջ միայն JavaScript և Node.js չի օգտագործվում։ Կարևոր է հասկանալ, որ որպես backend-ի լեզու կարող է օգտագործվել ցանկացած լեզու, իսկ հենց backend-ում լավ մասնագետ դառնալու համար անհրաժեշտ խորապես հասկանալ բոլոր կոնցեպտները, սկսած ցանցային ծրագրավորման հիմունքներից վերջացրած ինֆորմացիայի caching-ով։ Այս դասընթացը նախատեսվում է ծածկել բոլոր հնարավոր հարցերը, որոնց դուք կհանդիպեք թե՛ հարցազրույցին, թե՛ աշխատանքի ընթացքում։ Որպեսզի դառնալ օգտակար և արդյունավետ մասնագետ, իսկ շուկայում լինել նույնիսկ անփոխարինելի՝ անհրաժեշտ է տիրապետել մի քանի լեզուների, որոնք կիրառվում են backend-ում։ Հենց այդ է պատճառը, որ այս դասընթացում ուսուցանվում են Node.js (JavaScript), Java և Python։ Այս 8 ամիսների ընթացքում աշխատելու եք 4 մեծ նախագծերի վրա։ Սա աննախադեպ ինտենսիվ դասընթաց է, որի արդյունքում շուկայում կլինեք բարձրակարգ կրտսեր մասնագետներից (նույնիսկ mid-level)։
