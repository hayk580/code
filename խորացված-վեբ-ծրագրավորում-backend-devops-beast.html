Մանրամասներ

Յուրահատուկ դասընթաց, որն աշխատանքում հանդիպող լեզուները, տեխնոլոգիաներն ու հասկացությունները խորացված ուսումնասիրելուն զուգընթաց շեշտադրում է իրական նախագծերի վրա պրակտիկ աշխատանք։

Ամբողջ դասընթացի ընթացքում շաբաթական 4 օր տեսական դասերից (+1 օր անգլերենի դաս) բացի ամեն օր գրասենյակում աշխատելու եք վերը նշված հետաքրքիր, բարդ ու արտակարգ պորտֆոլիո հանդիսացող նախագծերի վրա։

Backend վեբ ծրագրավորման մեջ միայն JavaScript և Node.js չի օգտագործվում։ Կարևոր է հասկանալ, որ որպես backend-ի լեզու կարող է օգտագործվել ցանկացած լեզու, իսկ հենց backend-ում լավ մասնագետ դառնալու համար անհրաժեշտ խորապես հասկանալ բոլոր կոնցեպտները, սկսած ցանցային ծրագրավորման հիմունքներից վերջացրած ինֆորմացիայի caching-ով։

Backend ծրագրավորումը մի ամբողջ աշխարհ է։ Սովորաբար ուսանողներն, ովքեր սովորում են Node.js/Express ու մի փոքր MongoDB, ստանդարտ դասընթացներն ավարտելուց հետո բախվում են մի շարք խնդիրների՝ սկսած հարցազրույցը տապալելուց մինչև աշխատավայրում նորանոր խնդիրներին անարդյունավետ լուծումներ առաջարկելը։

Այս դասընթացում նախատեսվում է ծածկել բոլոր հնարավոր հարցերը, որոնց դուք կհանդիպեք թե՛ հարցազրույցին, թե՛ աշխատանքի ընթացքում։ Որպեսզի դառնալ օգտակար և արդյունավետ մասնագետ, իսկ շուկայում լինել նույնիսկ անփոխարինելի՝ անհրաժեշտ է տիրապետել մի քանի լեզուների, որոնք կիրառվում են backend-ում։ Հենց այդ է պատճառը, որ այս դասընթացում ուսուցանվում են Node.js (JavaScript), Java և Python։ Նպատակադրված ենք նաև ուսանողներին հնարավորություն տալ ընտրել .Net Core և Go՝ ըստ խիստ անհրաժեշտության։

Լեզուներից բացի, անհրաժեշտ է շատ լավ տիրապետել տվյալների մոդելավորման մոտեցումներին, տվյալների ճարտարապետությանը, ընդհանուր առումով հասկանալ թե ո՞ր դեպքում ի՞նչ տվյալների բազաներ օգտագործել, ի՞նչ տարբերություն ռելացիոն մոդելի ու NoSQL-ի միջև, ի՞նչ տեսակներ կան NoSQL համակարգերում և ո՞ր դեպքում որն օգտագործել, ի՞նչ տեսքով պահել տվյալները, ինչպե՞ս կազմակերպել համակարգի ընդլայնումը ըստ օգտագործողների թվի աճի, Ինչպե՞ս դարձնել համակարգն ավելի արդյունավետ՝ կիրառելով caching-ի տեխնոլոգիաներ, մասնատելով տվյալները, կիրառելով տեղաբաշխման ալգորիթմներ և այլն։ Վերջապես, ինչպե՞ս տեղադրել պատրաստի համակարգը սերվերում, ինչպե՞ս ավտոմատացնել build-ի և deployment-ի պրոցեսները, ինչու՞ կիրառել Docker, ե՞րբ օգտագործել Kubernetes և այլն։

Բոլոր նշված լեզուները, տեխնոլոգիաներն ու հասկացությունները կիրառվում են դասերին զուգահեռ պրակտիկ նախագծեր մշակելով և իրականացնելով։ Նախագծեր, որոնք կծածկեն ցանկացած գործատուի բոլոր հնարավոր կարիքներն ու պահանջները։

Դասընթացը իհարկե բարդ է, ինտենսիվ, սակայն պահանջվող ժամանակը տրամադրելով, դուք վերջապես կդառնաք բարձրակարգ մասնագետ ծրագրավորման, մասնավորապես՝ վեբ ծրագրավորման, ոլորտում։

Գիտահետազոտական կենտրոնի առավելությունները ուսումնական կենտրոնի նկատմամբ

  • Դասընթացներն այստեղ ֆորմալ բնույթի չեն, նպատակը պրակտիկ աշխատանքային փորձ ձեռք բերելն է
  • ՏՏ ընկերություններին JavaScript կամ C# կամ ցանկացած այլ լեզու սովորած ուսանողների փոխարեն անհրաժեշտ են խնդիրներ լուծողներ, ովքեր լեզվից անկախ են, ունեն Computer Science-ի ուժեղ ֆունդամենտալ գիտելիքներ և ստեղծել են որևէ միջին կամ մեծ նախագիծ ինքնուրույն
  • Բարձրորակ դասընթացների ցածր վարձավճարը նպատակաուղղված է պրակտիկ կրթությունը բոլորին հասանելի դարձնելուն
  • Աշխատում եք ամենաժամանակակից կամ բարդ ակադեմիական նախագծերի վրա, որոնք Ձեր ամբողջ կարիերայում կթողնեն մեծ ու օգտակար հետք
  • Անգլերենի դասերը պարտադիր են և ներառված են բոլոր դասընթացներում
  • Այստեղ չեք սովորում մեկ լեզու, մեկ տեխնոլոգիա կամ մեկ գործիք, այլ դառնում եք բարձրակարգ կրտսեր մասնագետ
  • Ցանկացած դասընթացին զուգահեռ կամ ավարտելուց հետո աշխատում եք իրական նախագծերի վրա աշխատանքային լուրջ պորտֆոլիո ձեռք բերելու համար
  • Պարբերաբար շփվում եք ոլորտի ուժեղագույն մասնագետների հետ, ովքեր աշխատում են տարբեր ՏՏ ընկերություններում
  • Հետազոտական աշխատանքների արդյունքում հեղինակում եք մեկ կամ մի քանի հոդվածներ
  • Նպատակը Ձեր տաղանդը բացահայտելն է, ոչ թե բիզնես մոդելի մասնիկ դարձնելը
  • Դառնում եք Հայաստանում առաջին ծրագրավորման գիտահետազոտական կենտրոնի անբաժան մաս

Դասընթացի ամբողջական ծրագիրը

Ստորև կգտնեք դասընթացի ծրագիրը՝ համառոտ վերնագրերով։ 8 ամիսների ընթացքում ունենալով շաբաթական չորս տեսական դաս՝ ընդհանուր ստացվում է 130 դաս։ Դասընթացի համառոտ նկարագրությունները վերաբերվում են միայն տեսական դասերին, իսկ զուգահեռ լինելու են գործնական աշխատանքներ, մեծ և փոքր նախագծերի իրականացումներ՝ սկսած 2-րդ ամսից։ Դասընթացը սկսվում է ծրագրավորման հիմունքների արագ ներածությամբ։ Դիտարկվում է հիմնականում C++ ծրագրավորման լեզուն, որը հիմք կհանդիսանա այլ լեզուները ավելի լավ ըմբռնելու համար։ Հիմնական համեմատությունները կլինեն Java լեզվի հետ, որին զուգահեռ կդիտարկվի նաև JavaScript-ը Node.js միջավայրում աշխատելու համար։ Այնուհետև backend վեբ ծրագրավորման բոլոր բաղկացուցիչները ամբողջ խորությամբ ուսումնասիրվում են տեսական և պրակտիկ դասերի ընթացքում։ Վեբ սերվերի ներքին կառուցվածք և աշխատանքի սկզբունքները, ցանցային ծրագրավորման հիմունքները, HTTP արձանագրությունը, քեշավորման սկզբունքներն ու ալգորիթմները, ծրագրային կոդի արխիտեկտուրա, տվյալների բազաների կառուցվածքն ու ներքին դետալները և բազմաթիվ այլ կարևոր թեմաներ, որոնք անբաժան են որակյալ backend ծրագրավորողի առօրյայից։

Դաս 1 - Ծրագրավորման լեզուներն ու կոմպիլացիայի մոդելները

Ներածություն դեպի ծրագրավորման լեզուներ, ցածր մակարդակից դեպի բարձր մակարդակ անցումը։ Բարձր մակարդակի լեզվով գրված ծրագրի թարգմանությունը ցածր մակարդակի լեզու։ Ծրագրի բազային կառուցվածքների ներկայացումը ցածր մակարդակում (պարզեցված assembly լեզվով)։ Ծրագրի մուտքի կետն ու կատարման հիմունքները կոմպիլացվող, ինտերպրետացվող և հիբրիդ լեզուների օրինակով։

Դաս 2 - Տվյալների տիպերը լեզուներում

Տվյալների տիպերն ու դրանց գոյության իմաստը։ C++ լեզվի հիմնական տվյալների տիպերն ու արժեքների ներկայացումը հիշողությունում։ Տվյալների գրել/կարդալու ինստրուկցիաները ցածր մակարդակի տեսանկյունից։ Ներածություն դեպի վիրտուալ հիշողություն։

Դաս 3 - Հիշողության սեգմենտներն ու հասցեավորման հիմունքները

Վիրտուալ հիշողության հիմունքներն ու ծրագրին հատկացվող հիշողության սեգմենտների նկարագրությունն ու իմաստը։ Վիրտուալ և ֆիզիկական հասցեների միջև կապն ու թարգմանությունը։ Հիշողության հասցեներ և աշխատանք հասցեավորվող փոփոխականների հետ։ C++ ցուցիչներ։

Դաս 4 - Դինամիկ հիշողություն

Դինամիկ հիշողության հատկացումը ծրագրին (memory allocation)։ Աշխատանք հատկացված հիշողության հետ ցուցիչների միջոցով։ Հիշողության ազատման սկզբունքներն ու ցածր մակարդակի դետալները։ Ներածություն դեպի garbage collector-ներ։

Դաս 5 - Ծրագրի կառուցվածքն ու պարունակությունը

Վերադարձ դեպի կոմպիլացվող լեզուներ։ C++ կոդի նախամշակում և կոմպիլացիա։ Օբյեկտային ֆայլեր և դրանց կապակցումը կատարվող ֆայլ ստանալու համար։ Linking-ի դետալները, symbol resolution, section relocation։ Ծրագրի բեռնումը դեպի հիշողություն։ Դինամիկ կապակցում։

Դաս 6 - Աբստրակցիայի մեխանիզմներ. struct

Ստրուկտուրաները C-ի տեսանկյունից։ ներկայացումը ցածր մակարդակում։ Անդամ փոփոխականներին դիմելու եղանակներն ու ցածր մակարդակում կազմակերպման դետալները։ Կլասների իմիտացիա ստրուկտուրաների միջոցով։

Դաս 7 - Աբստրակցիայի մեխանիզմներ. class

Կլասների իմաստն ու ֆիզիկական ներկայացումը ծրագրային կոդում։ Կլասների անդամ ֆունկցիաներ, անդամ ֆունկցիաների կանչը։ Աշխատանք օբյեկտների հետ։ Հիշողության դինամիկ հատկացում օբյեկտներին։ Օբյեկտների փոխանցումներ ֆունկցիաներին ըստ արժեքի և հղման։

Դաս 8 - Ժառանգում

Կլասների ժառանգման իմաստն ու ներկայացումը C-ով։ Ժառանգման տեսակներն ու այլընտրանքները։ Կոմպոզիցիա, աբստրակտ կլասներ, ինտերֆեյս։

Դաս 9 - Օբյեկտների և կլասների ներկայացումը JavaScript-ում

Օբյեկտները JavaScript-ում։ Կլասների հայտարարում և օգտագործումը։ JavaScript-ի կլասների իրականացման դետալները, hidden կլասների հասկացությունն ու կիրառումը։ Օբյեկտների փոխանցումը ֆունկցիաներին։ Հիշողության հատկացումը օբյեկտներին։

Դաս 10 - prototype ժառանգում

Ժառանգման մեխանիզմն ու իրականացումը JavaScript-ում։ Տարբերությունը C++-ի մոդելից։ prototype ժառանգման ներկայացումը ցածր մակարդակում։

Դաս 11 - Ասինխրոն կոդի կատարում

JavaScript-ի ինտերպրետացիայի ու կոդի կատարման հիմունքները։ Event loop-ի աշխատանքի սկզբունքները։ Node.js-ում libuv-ի օգտագործումն ու կոդի ասինխրոն կատարման նրբությունները։

Դաս 12 - JavaScript-ի նրբությունները

Տիպերի ներկայացումն ու փոխակերպումները։ Closure-ների իմաստն ու իրականացման ցածր մակարդակի դետալները։ Garbage collector-ի աշխատանքը հիշողության ազատման գործում։

Դաս 13 - ES6, ES.Next

JavaScript-ի ES6 թարմացումներն ու կիրառությունները։

Դաս 14 - Դաս 20 Node.js-ի լրացումները

Աշխատանք ֆայլերի հետ։ HTTP-ի մոդուլները։ Child պրոցեսներ։ Հոսքեր (streams)։ Այլ մոդուլներ։

Դաս 21 - Նախագծման խնդիրները մեծ նախագծերում (ներածություն)

Մեծ նախագծի ստեղծման փուլերն ու նախագծման նրբությունները։ Պահանջների հավաքագրում և բարձր մակարդակի արխիտեկտուրայի մշակում, կլասների բաժանում։ Ընթացող պրոցեսների դուրս բերում։ Ընդլայնման (scaling) նախապայմանների դիտարկում։

Դաս 22 - OOP հիմունքները

Օբյեկտ-կողմնորոշված ծրագրավորման հիմունքներն ու կիրառությունն իրական նախագծերում։ Նախագծման ընթացքում բարձր մակարդակի արխիտեկտուրայից դեպի կլասների դիագրամների ստացումն ու նախագծումը։ UML և դրա օգտագործումը նախագծման ընթացքում։ SOLID սկզբունքները։

Դաս 23 - Կլասների ինտերֆեյսի մշակում

Աբստրակտ կլասների և ինտերֆեյսների տարբերությունն ու օգտագործումը։ Նախագծման օրինակներ տարբեր մոտեցումներով։ Նախագծման ստուգարքային ցուցակի կազմում (checklist)։ Օբյեկտների փոխհաղորդակցության կետերն ու ինվարիանտների մշակումը։

Դաս 24 - Նախագծման շաբլոնների ներածություն

Design pattern-ների գաղափարն ու կիրառման նպատակները։ Ներածական օրինակներ։

Դաս 25 - Ալգորիթմական նախագծում

Նախագծման ցածր մակարդակի խնդիրներ, ալգորիթմական նախագծման ներածություն Autocomplete-ի օրինակով։ Տողերի որոնում, ներածություն դեպի ալգորիթմներ։

Դաս 26 - Ալգորիթմների ներածություն

Որոնման խնդրի օրինակով ներածություն դեպի ալգորիթմներ։ Գծային որոնում, բինար որոնում և տողերի ենթաորոնում։ Ալգորիթմների արագագործության չափում և վերլուծություն։ O(1), O(n), O(logn) և O(n^2) բարդության ալգորիթմների փոքր օրինակներ։

Դաս 27 - Տվյալների կառուցվածքների ներածություն

Տվյալների աբստրակցիաներ։ Պարզագույն տվյալների կառուցվածքներ (vector, linked list)։ Խնդիրների օրինակներ և տվյալների կառուցվածքների կիրառությունը դրանց լուծման համար։

Դաս 28 - Վեկտորի և կապակցված ցուցակի իրականացում

Վեկտորի և կապակցված ցուցակի կլասների նախագծում և գործողությունների իրականացում։ Գործողությունների բարդության համեմատականներ։ Հիբրիդ տվյալների կառուցվածքի նախագծում։ STL-ական կոնտեյներներ։

Դաս 29 - Gap Buffer

Դինամիկ վեկտորի կիրառությունը տեքստային խմբագրերի իրականացման մեջ։ Gap Buffer-ի նախագծում և իրականացում։

Դաս 30 - Տվյալների կառուցվածքների ադապտերներ

Stack և Queue տվյալների կառուցվածքների ադապտերներ։ Նախագծումն ու իրականացումը։ Ներածություն դեպի thread-safe Stack-ի նախագծում (կիրականացվի մի քանի դաս անց)։

Դաս 31 - Վեբ ծրագրավորման հիմունքները

Ի՞նչ բաղկացուցիչներ ունի վեբ ծրագրավորումը։ Ո՞րն է backend-ի դերը և ինչ է անհրաժեշտ որակյալ backend ծրագրավորող դառնալու համար։

Դաս 32 - Վեբ սերվերի կառուցվածքը

Վեբ սերվեր ծրագրի կառուցվածքն ու հիմնական դրույթները։ Nginx-ի նկարագրություն։

Դաս 33 - Համակարգչային ցանցեր

Ներածություն դեպի համակարգչային ցանցեր։ OSI մոդելի կառուցվածքը։

Դաս 34 - Ցանցային ծրագրավորման հիմունքներ

Ներածություն դեպի ցանցային ծրագրավորում։ Սոկետներ և դրանց օգտագործումը ցանցային ծրագրեր իրականացնելիս։

Դաս 35 - TCP սոկետ ծրագրավորում

TCP կլիենտ-սերվեր ծրագրերի նախագծում և իրականացում պարզագույն օրինակի վրա։

Դաս 36 - UDP սոկետ ծրագրավորում

UDP կլինետ-սերվեր ծրագրերի նախագծում և իրականացում պարզագույն օրինակի վրա։

Դաս 37 - Դաս 38 - HTTP արձանագրություն

HTTP արձանագրությունը, բաղկացուցիչներն ու դետալները։

Դաս 39 - Դաս 42 - C++ և ցանցային ծրագրավորում

Ցանցային ծրագրավորման կլասների նախագծում և մշակում (POSIX սոկետների հիման վրա)։ Պարզագույն վեբ սերվերի իրականացում սեփական մշակված ցանցային ծրագրավորման գրադարանի օգնությամբ։

Դաս 43 - Դաս 45 - Java և ցանցային ծրագրավորում

Վեբ սերվերի իրականացում Java-ի ցանցային ծրագրավորման միջոցներով։

Դաս 46 - Դաս 50 - Վեբ սոկետներ

Վեբ սոկետների ցածր մակարդակի իրականացման դետալները։ Սեփական հաղորդակցման արձանագրության նախագծում և մշակում։ Messenger ծրագրի նախագծում և իրականացում վեբ սոկետների հիման վրա։

Դաս 52 - Դաս 55 - REST API նախագծում

REST API նախագծման լավագույն օրինակներն ու մոտեցումները։ Node.js միջավայրում Express և Nest ֆրեյմվորկների օգտագործումը REST API տրամադրելու համար։ Java Spring ֆրեյմվորկի օգտագործումը REST API տրամադրման համար։

Դաս 56 - Authorization and authentication

API թոկենի օգտագործումը անվտանգության և նույնականացման համար։

Դաս 57 - JSON Web Token

JWT կիրառումը որպես API token և ընդլայնման օրինակներ։

Դաս 58 - Դաս 60 - OAuth 2 և իրականացումը սերվերում

OAuth 2-ի ներածություն և աշխատանքի սկզբունքները։ Սեփական սերվերում OAuth 2 նույնականացման իրականացում։

Դաս 61 - Դաս 63 - Permission-ների համակարգի նախագծում

Role-երի վրա հիմնված permission-ների ճկուն համակարգի նախագծում և իրականացում։

Դաս 64 - Տվյալների բազաների ներածություն

Ներածություն դեպի տվյալների պահպանում, տվյալների բազաներ և դրանց տեսակները։ Ռելացիոն և NoSQL լուծումների ներածություն։

Դաս 65 - Տվյալների ներկայացումը և պահպանումը ցածր մակարդակում

Համակարգչային հիշողությունը ցածր մակարդակում։ Ռեգիստրեր, cache հիշողություն, RAM և վիրտուալ հիշողություն։

Դաս 66 - Հիշող սարքերի տեսակները

Հիշող սարքերի կառուցվածքը ցածր մակարդակում։ RAM, DRAM, SRAM։

Դաս 67 - Տվյալների արդյունավետ պահպանումը տվյալների կառուցվածքներում

Բալանսավորված բինար որոնման ծառ։ AVL ծառի և RB-ծառի օրինակով։

Դաս 68 - Բալանսավորված ծառերի իրականացում

RB-ծառի իրականացում։ Ներածություն դեպի 2-3 և 2-3-4 ծառեր։

Դաս 69 - B-ծառեր

B-trees, ներածություն և կիրառության օրինակներ։ Կիրառությունը տվյալների բազաների համակարգերում։

Դաս 73 - Դաս 76 - Ռելացիոն տվյալների բազաներ

Ռելացիոն տվյալների բազաների կառուցվածքը։ Տվյալների պահպանման սկզբունքները ռելացիոն տվյալների բազաներում։ Tuple-ներ և ներկայացումը հիշողության մեջ։

Դաս 77 - Դաս 79 - Ռելացիոն տվյալների բազաների կիրառությունները

Ռելացիոն տվյալների բազաների կիրառության օրինակներ։ Օգտագործման տարբերակներն ու օրինակները։ SQL հարցումների լեզվի ներածություն։

Դաս 80 - Դաս 82 - Տվյալների ճարտարապետություն

Տվյալների բաշխման և ներկայացման մեթոդներ ու մոտեցումներ։ Նորմալիզացիայի կանոնները ռելացիոն տվյալների բազաներում։

Դաս 83 - Դաս 85 - Տվյալների ճարտարապետություն (օրինակ)

Տվյալների կազմակերպման և ներկայացման մոտեցումները տարբեր նախագծերի օրինակներով (ռելացիոն տվյալների բազաների շեշտադրությամբ)։

PostgreSQL համակարգի ներածություն, կիրառություն և հարցումների օրինակներ։

Դաս 94 - Դաս 100 - MongoDB օգտագործումը

Դաս 94 - Դաս 100 - MongoDB օգտագործումը MongoDB օգտագործումն ու համեմատականները PostgreSQL-ի հետ։

Ինդեքսավորման ցածր մակարդակի դետալները։ Ինդեքսի կառուցվածքն ու իրականացումը։

Դաս 106 - Դաս 110 Ինդեքսավորումը որոնողական համակարգերում

Որոնողական համակարգի նախագծում։ Որոնողական համակարգերի կառուցվածքի ու աշխատանքի սկզբունքները։ Elasticsearch-ի օգտագործման օրինակներ, որոնողական համակարգի ինդեքսի կառուցման մոտեցումներ։

Դաս 111 - Դաս 115 - Տվյալների քեշավորում

Քեշավորման հիմունքները, մոտեցումներն ու ալգորիթմները։ Քեշավորումը վեբ սերվերում։ Քեշ համակարգերի ներդրում, օգտագործում (Memcached, Redis)։
