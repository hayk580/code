Code Republic-ը ստեղծված է էնտուզիաստների համար

Ծրագրավորողների, ինժեներների, մաթեմատիկոսների և ֆիզիկոսների համար

  • Տեսաձայնագրել որակյալ դասընթացներ մեր ստուդիայում, դասընթացները տեղադրում ենք մեր YouTube-յան ալիքում (կարող եք նույնիսկ հղում կատարել սեփական ալիքի, ուսումնական կենտրոնի կամ ընկերության վրա)
  • Գրել տեխնիկական հոդված կամ հոդվածներ Ձեզ նախընտրելի թեմայով
  • Ղեկավարել ուսանողական նախագծերը (հեշտ նախագծեր չեն)
  • Կազմակերպել և ղեկավարել հետազոտական աշխատանքներ, որի արդյունքում կհրապարակվեն հոդվածներ արտասահմանյան ամսագրերում
  • Դասավանդել Code Republic-ում (վճարվող աշխատանք է, դասախոսների ընտրությանը շատ խիստ ենք մոտենում)
  • Դասավանդել մաթեմատիկա, ֆիզիկա, և/կամ mechanical engineering-ի ցանկացած ստորաբաժանում (խորը գնացող պլաններ ունենք)
  • Code review-ի ենթարկել ուսանողների կատարած առաջադրանքները
  • Տպագրվել որպես հայալեզու տեխնիկական գրքի հեղինակ կամ համահեղինակ (խմբագրման, վերստուգման և տպագրման աշխատանքները կատարվում են Code Republic-ում)
  • Հավաքագրել ուսանողներից բաղկացած թիմ սեփական նախագծերի համար

Այլ մասնագետների համար

  • Կատարել գրաֆիկական աշխատանքներ ուսումնական նյութերի պատրաստման համար
  • Օգնել առցանց անվճար վիդեո դասընթացների մոնտաժով
  • Տարածել Code Republic-ը սոցիալական հարթակներում (գտեք մեզ LinkedIn, Facebook, Medium, YouTube, Instagram)
  • Շփվել ուսանողների հետ անգլերենով (ուսանողները ամեն դեպքում անցնում են անգլերեն դասեր, սակայն լրացուցիչ պրակտիկան երբեք չի խանգարի)
  • Այլ կերպ (գրեք մեզ contact@coderepublic.am)

Մանրամասներ

Code Republic-ը ստեղծվել է էնտուզիաստների համար էնտուզիաստների կողմից։ Իհարկե ունենք բիզնես մոդել, ունենք վճարովի դասընթացներ, սակայն այն ամբողջ ներուժը, որ կատարում ենք վճարովի դասընթացներում ի համեմատական վարձավճարի հետ, ստացվում է ավելին քան անվճար։ Մենք ձգտում ենք պատրաստել երկրի մասշտաբով լավագույն կադրերին, ուսանողների ընտրությանը մոտենում ենք զգուշորեն, ուսանողները գտնվում են գրասենյակում ամեն օր 4-6 ժամով և աշխատում են պրակտիկ նախագծերի վրա։ Մասնակցում են խորացված տեսական դասերին շաբաթական չորս անգամ 2-ական ժամով, իսկ շաբաթական մեկ անգամ մասնակցում են անգլերեն դասերին։ Ամեն ամիս հրավիրում ենք ՏՏ ընկերությունների ներկայացուցիչներին ուսանողների հետ հարցազրույցներ անցկացնելու համար։ Այս ամբողջը միասին ցածր, ամսական ընդամենը 42 հազար դրամ վարձավճարով, որի նպատակն է ուսանողի կենտրոնում գտնվելու հարմարավետությունը բարձր մակարդակի վրա պահելը։

Սա աննախադեպ, ինչ-որ տեղ նաև անիրական թվացող ընկերություն է, բայց դեռ ամենը չէ։ Մենք պարբերաբար տեսաձայնագրում ենք խորացված վիդեո դասընթացներ ու անվճար տեղադրում YouTube-յան ալիքում։ Գրում ենք կամ թարգմանում տեխնիկական հոդվածներ, գրում ենք գրքեր, որոնք տպագրելուց հետո տարածելու ենք անվճար բոլորի համար։ Եվ այդ ամենի հետ մեկտեղ, համակարգում ենք ու կազմակերպում մեզ նման էնտուզիաստների աշխատանքը, որ էլ ավելի հսկա տեմպերով զարգացնենք մեր սկզբնական գաղափարը։

Իսկ գաղափարը պարզ է, ստեղծել հարթակ բոլոր մասնագետների համար՝ նպաստելով երկրի գիտահետազոտական ներուժի իրական, օգտակար, արդյունավետ աճին։ Մենք սիրում ենք գործել, միացե՛ք մեզ։
